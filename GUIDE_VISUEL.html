    Ajoute tes icônes

    2 min
    1. Va dans le dossier : public/icons/
    2. Copie tes 2 images de logo
    3. Renomme-les :
      Grande image → icon-512.png
      Moyenne image → icon-192.png
    2

    Push sur GitHub

    5 min

    Option A : Git Bash

    git init
    git add .
    git commit -m "Fix: Add React and Vercel config"
    git remote add origin https://github.com/Kbg-boa/usa-level-english.git
    git branch -M main
    git push -u origin main

    Option B : GitHub Desktop (plus simple)

    1. Télécharge : https://desktop.github.com/
    2. Installe et connecte-toi
    3. File → Add Local Repository
    4. Sélectionne le dossier
    5. Publish repository
    3

    Déploie sur Vercel

    3 min
    1. Va sur : https://vercel.com
    2. Clique "Continue with GitHub"
    3. Connecte-toi
    4. Clique "Add New..." → "Project"
    5. Clique "Import" à côté de ton repo
    6. Laisse tout par défaut
    7. Clique "Deploy"
    8. Attends 2-3 minutes...
    9. 🎉 URL prête : https://usa-level-english.vercel.app
    4

    Installe sur iPhone

    1 min
    1. Ouvre Safari (pas Chrome !)
    2. Va sur l'URL Vercel
    3. Clique "Partager" (bouton avec flèche)
    4. Clique "Sur l'écran d'accueil"
    5. Clique "Ajouter"
    6. 🎉 L'icône US apparaît !

    📊 Résumé

    Après avoir suivi ces 4 étapes, ton app sera en ligne et installable sur iPhone !
