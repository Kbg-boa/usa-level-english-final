## 🎨 GÉNÉRATEUR D'ICÔNES

### **`/public/icons/generate-icons.html`** ⭐ **FICHIER PRINCIPAL**

**C'est quoi ?**
- Une interface web magnifique
- Génère automatiquement les 2 icônes PNG dont tu as besoin
- Design premium : étoile USA + texte "USA" + gradient bleu/rouge

**Comment l'utiliser ?**
1. Ouvre ce fichier dans un navigateur (double-clic)
2. Clique "Télécharger les 2 icônes"
3. Les fichiers PNG sont prêts dans tes Téléchargements !

**Ce qu'il fait :**
- Affiche 2 aperçus des icônes (192×192 et 512×512)
- Génère les PNG via Canvas (qualité maximale)
- Télécharge en 1 clic
- Design 100% cohérent avec l'app
