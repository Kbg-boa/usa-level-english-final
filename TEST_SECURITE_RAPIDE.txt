MÉTHODE 2: TEST MANUEL (2 MIN)
=============================================================

1. VA SUR TON SITE
-------------------------------------------------------------
https://ton-site.vercel.app


2. OUVRE DEVTOOLS (F12)
-------------------------------------------------------------
Windows: F12
Mac: Cmd+Option+I


3. NETWORK → REFRESH PAGE
-------------------------------------------------------------
- Clic sur première requête (ton site)
- Onglet "Headers"
- Scroll → "Response Headers"


4. VÉRIFIE CES HEADERS ✅
-------------------------------------------------------------
Tu DOIS voir:

✅ content-security-policy: default-src 'self'...
✅ strict-transport-security: max-age=31536000...
✅ x-frame-options: DENY
✅ x-content-type-options: nosniff
✅ x-xss-protection: 1; mode=block

Si tu vois TOUS ces headers → Niveau 1 actif ! ✅


5. TEST BONUS: DONNÉES PROTÉGÉES
-------------------------------------------------------------
DevTools → Sources (ou Debugger)
  → Cherche fichier avec "vocabulary"
  → Regarde le contenu

Tu vois:
  ❌ export const vocabularyDatabase = { ... 9000 mots }
  → Données STATIQUES (normal pour maintenant)

Après Supabase, tu verras:
  ✅ export async function getVocabulary() { ... }
  → Données PROTÉGÉES (backend)


6. TEST ONLINE (OPTIONNEL)
-------------------------------------------------------------
https://securityheaders.com

  1. Entre ton URL
  2. Clic "Scan"
  3. Regarde le grade

Tu devrais avoir:
  🟢 Grade A ou B (40-50/100)
