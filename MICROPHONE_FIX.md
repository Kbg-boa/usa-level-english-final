# 🎤 FIX MICROPHONE ERROR: "not-allowed"

## ❌ ERREUR: "Voice recognition error: not-allowed"

**Cause**: Ton navigateur BLOQUE l'accès au microphone

---

## ✅ SOLUTION RAPIDE (3 étapes)

### **Étape 1**: Autorise le Microphone

**Chrome / Edge** (Recommandé):

1. Regarde la **barre d'adresse** (en haut)
2. Tu vois l'icône **🔒** ou **🛡️**?
3. **CLIQUE** dessus
4. Cherche **"Microphone"** dans la liste
5. Change de **"Bloquer"** ou **"Demander"** → **"Autoriser"**
6. Ferme le popup

**Safari** (Mac):

1. Menu **Safari** → **Préférences**
2. Onglet **"Sites web"**
3. Dans la liste de gauche → **"Microphone"**
4. Trouve ton site dans la liste
5. Change le menu déroulant → **"Autoriser"**

**Safari** (Version rapide):

1. Barre d'adresse → Clique sur **"aA"** ou icône du site
2. Sélectionne **"Réglages pour ce site web"**
3. Microphone → **"Autoriser"**

---

### **Étape 2**: Recharge la Page

**Méthodes**:
- Appuie sur **F5**
- OU **Ctrl + R** (Windows/Linux)
- OU **Cmd + R** (Mac)
- OU clique le bouton de rechargement ↻

---

### **Étape 3**: Teste le Microphone

1. Va sur `/conversation`
2. Démarre une conversation (3-Min Sprint)
3. **Clique** le bouton **🎤 VIOLET**
4. Une popup apparaît: **"Autoriser l'accès au microphone?"**
5. **CLIQUE "AUTORISER"** ✅
6. Le bouton devient **ROUGE** et pulse
7. **PARLE**: "Testing one two three"
8. Le texte devrait apparaître!

---

## 🔧 AUTRES PROBLÈMES POSSIBLES

### **"Microphone not found"**

**Cause**: Pas de micro détecté

**Solutions**:
1. ✅ Microphone est branché (USB, casque)?
2. ✅ Fonctionne dans Zoom, Discord, Teams?
3. ✅ Pas utilisé par une autre application?
4. ✅ Dans Paramètres Windows/Mac → Son → Micro activé?

**Windows**:
```
Paramètres → Système → Son
→ Entrée → Choisis ton microphone
→ Test avec la barre de volume
```

**Mac**:
```
Préférences Système → Son
→ Entrée → Sélectionne ton microphone
→ Parle → les barres bougent?
```

---

### **"No speech detected"**

**Cause**: Le micro n'entend rien

**Solutions**:
1. ✅ Parle plus FORT
2. ✅ Microphone plus PROCHE (5-10cm)
3. ✅ Vérifie que le micro n'est pas en MUTE
4. ✅ Teste avec une autre app (Enregistreur vocal)
5. ✅ Réduis le bruit de fond

---

### **Le texte ne correspond pas à ce que tu dis**

**Cause**: Reconnaissance vocale imprécise

**Solutions**:
1. ✅ Parle plus **CLAIREMENT**
2. ✅ Prononciation **CORRECTE** (accent américain)
3. ✅ Parle **MOINS VITE**
4. ✅ Environnement **CALME**
5. ✅ Utilise un **CASQUE-MICRO** (meilleure qualité)

**Note**: Les erreurs de reconnaissance sont NORMALES!
- C'est une opportunité d'améliorer ta prononciation!
- Corrige le texte avant d'envoyer

---

### **Ça marchait avant, maintenant ça ne marche plus**

**Solutions rapides**:

```
1. Recharge la page (F5)
2. Ferme l'onglet et rouvre
3. Redémarre le navigateur
4. Vérifie que le micro fonctionne ailleurs
5. Vide le cache navigateur
6. Réinitialise les permissions (voir ci-dessus)
```
