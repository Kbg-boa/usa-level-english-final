## 🚀 MÉTHODE RAPIDE (5 MINUTES)

### **Option 1 : Générateur automatique**

1. Va sur [favicon.io/favicon-generator](https://favicon.io/favicon-generator/)
2. Configure :
   - **Texte** : `🇺🇸`
   - **Background** : `#0B1220`
   - **Font** : Bold
   - **Font Size** : 110
3. Clique "Download"
4. Extrait le ZIP
5. Renomme les fichiers :
   - `android-chrome-192x192.png` → `icon-192.png`
   - `android-chrome-512x512.png` → `icon-512.png`
6. Place-les dans ce dossier `/public/icons/`

---

### **Option 2 : Design personnalisé**

#### **Avec Canva :**

1. Va sur [canva.com](https://canva.com)
2. Crée un design personnalisé 512x512
3. Design suggéré :
   ```
   Fond : Dégradé (#0B1220 → #1E3A8A)
   Icône : 🇺🇸 ou "USA"
   Texte : "LEVEL" en petit
   ```
4. Télécharge en PNG (512x512)
5. Redimensionne une copie à 192x192
6. Place les 2 fichiers ici

#### **Avec Figma :**

1. Crée un frame 512x512
2. Ajoute ton design
3. Export :
   - `icon-512.png` @ 1x
   - `icon-192.png` @ 0.375x (ou resize après)
4. Place les fichiers ici
