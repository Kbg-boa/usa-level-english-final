## 1️⃣ CRÉER LES ICÔNES

### **🎯 MÉTHODE ULTRA-RAPIDE**

Va sur [favicon.io/favicon-generator](https://favicon.io/favicon-generator/)

**Configuration :**
```
Texte : 🇺🇸
Background : #0B1220
Font : Bold
Font Size : 110
```

**Télécharge et renomme :**
```
android-chrome-192x192.png  →  icon-192.png
android-chrome-512x512.png  →  icon-512.png
```

**Place dans :**
```
/public/icons/icon-192.png
/public/icons/icon-512.png
```

✅ **Fait !**

---

## 2️⃣ PUSH SUR GITHUB

### **Terminal (copie-colle) :**

```bash
# Init Git
git init
git add .
git commit -m "USA Level English - Production ready"
git branch -M main
```

### **Sur GitHub :**

1. Va sur [github.com/new](https://github.com/new)
2. Nom : `usa-level-english`
3. Clique "Create repository"

### **Retour Terminal :**

```bash
# Remplace TON_USERNAME par ton vrai username GitHub
git remote add origin https://github.com/TON_USERNAME/usa-level-english.git
git push -u origin main
```

✅ **Code en ligne !**

---

## 3️⃣ DÉPLOYER SUR VERCEL

### **Sur Vercel :**

1. Va sur [vercel.com](https://vercel.com)
2. Sign up avec GitHub
3. "Add New..." → "Project"
4. Sélectionne `usa-level-english`
5. Clique "Deploy"

⏳ **Attends 2-3 minutes...**

✅ **APP EN LIGNE !**

Tu reçois un lien comme :
```
https://usa-level-english.vercel.app
```

---

## 📱 INSTALLER SUR TON TÉLÉPHONE

### **iPhone :**
1. Ouvre Safari
2. Va sur ton lien Vercel
3. "Partager" → "Sur l'écran d'accueil"

### **Android :**
1. Ouvre Chrome
2. Va sur ton lien Vercel
3. Menu → "Ajouter à l'écran d'accueil"
