## **ÉTAPE 1 : AJOUTE TES ICÔNES (2 MIN)**

## **ÉTAPE 2 : PUSH SUR GITHUB (5 MIN)**

### **Option A : Avec Git Bash (si tu as Git)**

**Ouvre Git Bash dans le dossier du projet** (clic droit → "Git Bash Here")

**Copie-colle ces commandes UNE PAR UNE :**

```bash
git init
```

```bash
git add .
```

```bash
git commit -m "Fix: Add React and Vercel config"
```

**Maintenant va sur GitHub.com :**
1. Clique le bouton **"+"** en haut à droite
2. Clique **"New repository"**
3. Nom : `usa-level-english`
4. **NE COCHE PAS** "Initialize with README"
5. Clique **"Create repository"**

**GitHub te montre des commandes, IGNORE-LES et tape ceci dans Git Bash :**

```bash
git remote add origin https://github.com/Kbg-boa/usa-level-english.git
```

```bash
git branch -M main
```

```bash
git push -u origin main
```

**✅ Ton code est sur GitHub !**

---

### **Option B : Avec GitHub Desktop (plus simple)**

1. **Télécharge GitHub Desktop :**
   ```
   https://desktop.github.com/
   ```

2. **Installe et connecte-toi**

3. **Clique "File" → "Add Local Repository"**

4. **Sélectionne ton dossier dézippé**

5. **Si ça dit "not a git repository", clique "Create repository"**

6. **Remplis :**
   - Name: `usa-level-english`
   - Description: `American English learning app`
   
7. **Clique "Create Repository"**

8. **Clique "Publish repository"**

9. **Décoche "Keep this code private" (pour que Vercel puisse y accéder gratuitement)**

10. **Clique "Publish"**

**✅ Ton code est sur GitHub !**

---

## **ÉTAPE 3 : DÉPLOYER SUR VERCEL (3 MIN)**

1. **Va sur :**
   ```
   https://vercel.com
   ```

2. **Clique "Continue with GitHub"**

3. **Connecte-toi avec ton compte GitHub**

4. **Clique "Add New..." → "Project"**

5. **Tu vois ton repo `usa-level-english`**

6. **Clique "Import"**

7. **Vercel remplit automatiquement :**
   - Framework: Vite ✅
   - Build Command: `npm run build` ✅
   - Output Directory: `dist` ✅

8. **Laisse tout par défaut**

9. **Clique "Deploy"**

10. **Attends 2-3 minutes... ⏳**

11. **🎉 VERCEL TE DONNE UNE URL !**
    ```
    https://usa-level-english.vercel.app
    ```

**✅ TON APP EST EN LIGNE ! 🚀**

---

## **ÉTAPE 4 : INSTALLE SUR TON iPHONE (1 MIN)**

**Sur ton iPhone :**

1. **Ouvre Safari** (PAS Chrome !)

2. **Va sur l'URL Vercel :**
   ```
   https://usa-level-english.vercel.app
   ```

3. **L'app s'ouvre dans Safari**

4. **Clique le bouton "Partager"** (carré avec flèche vers le haut)

5. **Scroll vers le bas et clique "Sur l'écran d'accueil"**

6. **Clique "Ajouter"**

7. **L'icône avec ton logo US apparaît sur l'écran d'accueil ! 📱**

8. **Clique dessus → L'app s'ouvre en plein écran !**

**✅ TERMINÉ ! TON APP EST SUR TON iPHONE ! 🎉**
