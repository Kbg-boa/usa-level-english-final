═══════════════════════════════════════════════════════════════
  ÉTAPE 2 : PUSH SUR GITHUB (5 MIN)
═══════════════════════════════════════════════════════════════

┌─ OPTION A : GIT BASH (si tu as Git installé) ────────────┐
│                                                            │
│  1. Clic droit dans le dossier → "Git Bash Here"         │
│                                                            │
│  2. Copie-colle ces commandes UNE PAR UNE :              │
│                                                            │
│     git init                                               │
│     git add .                                              │
│     git commit -m "Fix: Add React and Vercel config"      │
│                                                            │
│  3. Va sur github.com :                                    │
│     - Clique "+" → "New repository"                        │
│     - Nom : usa-level-english                              │
│     - NE COCHE PAS "Initialize with README"                │
│     - Clique "Create repository"                           │
│                                                            │
│  4. Retourne dans Git Bash et tape :                       │
│                                                            │
│     git remote add origin https://github.com/Kbg-boa/usa-level-english.git  │
│     git branch -M main                                     │
│     git push -u origin main                                │
│                                                            │
│  ✅ Sur GitHub !                                          │
│                                                            │
└────────────────────────────────────────────────────────────┘

┌─ OPTION B : GITHUB DESKTOP (plus facile) ─────────────────┐
│                                                            │
│  1. Télécharge : https://desktop.github.com/             │
│                                                            │
│  2. Installe et connecte-toi avec ton compte GitHub      │
│                                                            │
│  3. Clique "File" → "Add Local Repository"               │
│                                                            │
│  4. Sélectionne le dossier du projet                      │
│                                                            │
│  5. Si demandé, clique "Create Repository"                │
│                                                            │
│  6. Clique "Publish repository"                           │
│                                                            │
│  7. Décoche "Keep this code private"                      │
│                                                            │
│  8. Clique "Publish"                                      │
│                                                            │
│  ✅ Sur GitHub !                                          │
│                                                            │
└────────────────────────────────────────────────────────────┘


═══════════════════════════════════════════════════════════════
  ÉTAPE 3 : DÉPLOIE SUR VERCEL (3 MIN)
═══════════════════════════════════════════════════════════════

1. Va sur : https://vercel.com

2. Clique "Continue with GitHub"

3. Connecte-toi

4. Clique "Add New..." → "Project"

5. Clique "Import" à côté de "usa-level-english"

6. Laisse TOUT par défaut

7. Clique "Deploy"

8. Attends 2-3 minutes... ⏳

9. 🎉 VERCEL TE DONNE UNE URL !
   
   https://usa-level-english.vercel.app

✅ TON APP EST EN LIGNE ! 🚀


═══════════════════════════════════════════════════════════════
  ÉTAPE 4 : INSTALLE SUR TON iPHONE (1 MIN)
═══════════════════════════════════════════════════════════════

1. Sur ton iPhone, ouvre SAFARI (pas Chrome !)

2. Va sur : https://usa-level-english.vercel.app

3. Clique le bouton "Partager" (carré avec flèche)

4. Scroll vers le bas

5. Clique "Sur l'écran d'accueil"

6. Clique "Ajouter"

7. L'icône avec ton logo US apparaît ! 📱

8. Clique dessus → L'app s'ouvre en plein écran !

✅ TON APP EST SUR TON iPHONE ! 🎉
